This module generates a summary report of all components used in the
selected manufacturing orders. Aggregates the quantities from all the
orders and shows summary using the following format:

============== ======== =====
Component Name Quantity UoM
============== ======== =====
Apples         20       Items
Bananas        3        Items
============== ======== =====

Important: currently this module shows only the top-level components.

Use Cases / Context
===================

This module addresses the challenge of efficiently aggregating
components from multiple manufacturing orders in Odoo. Standard Odoo
lacks a direct method to consolidate component lists, making it
difficult to assess total material needs, identify common components, or
generate summarized kitting lists, often leading to manual and
error-prone processes.

Usage
=====

- Navigate to **Manufacturing ▸ Operations ▸ Manufacturing Orders**.
- Select the orders you want to report on.
- Click **Print** ▸ **Component Report**.

An XLSX file will be downloaded.
